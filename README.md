# Chathura Fernando — Portfolio

Personal portfolio for Chathura Fernando, PCB Designer & Embedded Systems Engineer at Atlas Labs.

## Stack

- **Vite** + **React**
- **Tailwind CSS v4** (`@tailwindcss/vite`)
- **Framer Motion**

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# output → dist/
```

Deploy via Vercel — `vercel.json` is pre-configured with SPA rewrites and cache headers.

## CV

`public/Chathura_Fernando_CV.pdf` is served at `/Chathura_Fernando_CV.pdf` and linked from both the hero and contact sections.
